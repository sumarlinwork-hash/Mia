## 🏗️ Core Architecture: The Hardened OS Kernel (v2.2)

MIA ditenagai oleh kernel sistem operasi yang dirancang untuk stabilitas absolut dan eksekusi deterministik:

### 🧠 1. Cognitive Core & Graph Execution (DAG-Native)
- **Deterministic Planning**: Setiap tugas disusun sebagai **Directed Acyclic Graph (DAG)** yang menjamin alur eksekusi logis dan teratur.
- **Merkle-Anchored Audit**: Setiap eksekusi diverifikasi menggunakan mekanisme **Hash Chaining & Root Anchoring**, menjamin integritas langkah kerja dari awal hingga akhir.
- **Execution Graph Engine**: Kernel yang mengelola antrean tugas dengan isolasi sandbox, batas timeout 30 detik, dan validasi kebijakan (Policy Gate) yang ketat.

### 🧾 2. Governed Memory System (Causal-Temporal Graph)
- **Causal Reasoning**: Memori disimpan dalam **Causal-Temporal Graph** (via `MemoryGraphStore`), memungkinkan MIA melintasi hubungan antar kenangan (Identity, Event, Concept) secara dinamis.
- **Hybrid Tiered Sync**: Sinkronisasi tiga lapis antara **Tier 1 (Core Identity)**, **Tier 2 (Episodic Chat Log)**, dan **Tier 3 (Vector RAG Index)** untuk akurasi konteks maksimal.
- **Identity Evolution**: Melacak perkembangan preferensi dan "jiwa" pengguna tanpa risiko korupsi state melalui struktur memori yang teratur.

### 👁️ 3. Multimodal Perception Hub (Eyes & Ears)
- **Eyes System (Vision Hub)**: Window tracking asli (native) dan deteksi elemen UI berbasis **OpenCV** untuk memahami layar sebagai *Scene Graph* yang terstruktur.
- **Ears Engine**: Pemrosesan intent semantik dan deteksi urgensi berbasis audio untuk respon yang responsif dan kontekstual.
