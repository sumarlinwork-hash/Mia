# PRD - Discovery Marketplace

## Vision
Membuat Discovery sebagai pusat instalasi kemampuan AI (plugin, MCP server, tools) dengan pengalaman setara marketplace modern: cepat ditemukan, aman dipasang, jelas statusnya, mudah dioperasikan.

## Primary Users
- Solo builder
- Developer power user
- Business operator
- Content creator

## Core Jobs To Be Done
- Menemukan tools sesuai use-case dengan cepat.
- Memahami trust, permission, dan harga sebelum install.
- Memasang, update, menjalankan, atau uninstall tanpa friction.
- Mempublikasikan item dan mengelola release.

## Functional Requirements
1. Catalog Service:
   - Menyediakan daftar item lintas kategori.
2. Search + Ranking:
   - Query, filter, ranking score.
3. Manifest Standard:
   - `id`, `version`, `author`, `capabilities`, `required_permissions`, `pricing`, `rating`, `compatibility`.
4. Install Orchestrator:
   - Install/uninstall source-aware.
5. Permission Consent:
   - Consent wajib untuk permission berisiko tinggi.
6. Update Manager:
   - Deteksi update dan one-click upgrade.
7. Publisher Console:
   - Submit, review, approve/reject, publish release.
8. Audit Log:
   - Jejak semua aksi kritikal.

## Non-Functional Requirements
- Responsif pada daftar besar.
- Mudah diperluas ke monetization.
- Data state persisten.
- Struktur modular agar maintenance ringan.

## Success Metrics
- Conversion discovery-to-install.
- Install success rate.
- Update adoption rate.
- Publisher approval lead-time.
- Retensi 7-day installed tools.
