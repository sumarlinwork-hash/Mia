function dispatch_execution(user, skill_id, input):

if not can_execute(user, skill_id):
return error("license invalid")

task = {
id: uuid(),
skill_id: skill_id,
input: input,
status: "queued",
retry: 0,
created_at: now()
}

queue.push(task)

return task.id

---

## 4.2 WORKER LOOP

while true:

task = queue.pop()

if task == null:
sleep(100ms)
continue

try:

```
result = run_skill(
  task.skill_id,
  task.input,
  timeout = 30s
)

store_result(task.id, result)
mark_complete(task.id)
```

except TimeoutError:

```
mark_timeout(task.id)
```

except Exception as err:

```
retry_or_fail(task, err)
```

---

## 4.3 RETRY ENGINE

function retry_or_fail(task, err):

if task.retry < 3:

```
task.retry += 1
queue.push(task)
```

else:

```
mark_failed(task.id, err)
```
