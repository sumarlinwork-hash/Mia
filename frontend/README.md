# 🎨 MIA Frontend: The Interactive Soul

Antarmuka MIA dirancang dengan estetika **Premium Dark Architecture**, mengutamakan imersi, responsivitas, dan kedekatan emosional (Intimacy UX). Ditenagai oleh React + Vite + TypeScript.

---

## 🌟 Fitur Utama Frontend

### 💓 1. Aura Interactive Engine
Pusat visualisasi MIA di halaman utama (`Home.tsx`).
- **Heartbeat Visualization**: Ikon aktivitas yang berdenyut sinkron dengan detak jantung digital MIA.
- **Resonant Skin**: Komponen aura yang merespons sentuhan kursor (hover/click) untuk memicu reaksi emosional.
- **Audio Visualizer**: Bar frekuensi dinamis yang muncul saat MIA berbicara, terintegrasi dengan Web Audio API.

### ⌨️ 2. Unified Command Palette (`Ctrl+K`)
Pusat navigasi dan aksi cepat yang dapat diakses dari mana saja.
- **Quick Toggles**: Aktivasi instan untuk *Soulmate Mode*, *Zen Mode*, dan pembersihan memori.
- **Action Search**: Menjalankan perintah sistem atau menavigasi ke pengaturan tanpa meninggalkan keyboard.

### 🛡️ 3. Crone Control Center (`Crone.tsx`)
Dashboard pemantauan tugas latar belakang.
- **Manual Overrides**: Tombol Start/Pause/Resume untuk setiap job.
- **Cost Transparency**: Indikator biaya token (Low/High) untuk manajemen kuota API yang cerdas.

### 🎨 4. Dynamic UI States
Antarmuka yang berubah secara dramatis berdasarkan konteks:
- **Intimacy Phase**: Transisi warna ke Rose Pink, efek *Cinema Mode* (vignette), dan pendaran aura yang hangat.
- **Zen Mode**: Privasi total dengan menyembunyikan UI dan memfokuskan layar hanya pada wallpaper/aura minimalis.
- **Mood Overlays**: Lapisan warna dinamis (Energetic/Serious/Warm) berdasarkan sentimen pesan MIA.

---

## 🛠 Tech Stack & Architecture

- **Core**: React 18 (Hooks-based), TypeScript.
- **Build Tool**: Vite (Ultra-fast HMR).
- **Styling**: 
  - **TailwindCSS v4**: Utilitas modern untuk tata letak responsif.
  - **Custom CSS Keyframes**: Untuk animasi kompleks seperti `heartbeat` dan `chat-spring`.
- **Icons**: Lucide React.
- **Markdown**: React-Markdown + Prism (untuk rendering kode).

---

## 🚀 Jalur Pengembangan (Dev Workflow)

### Menjalankan secara Lokal:
```bash
npm run dev
```
Pastikan backend MIA berjalan di port `8000` agar seluruh fitur interaktif (WS & API) dapat berfungsi dengan normal.
