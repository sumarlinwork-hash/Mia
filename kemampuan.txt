# ❗ CELAH YANG MASIH ADA (KRITIS)

## ⚠️ ISSUE-1: HASH COVERAGE BELUM JELAS

Sekarang:

```json
entry_hash = ?
```

Saya belum lihat Anda mengikat:

```text
path + blob + seq + prev_hash
```

### Kalau tidak lengkap:

👉 attacker / corruption bisa:

* ganti `path`
* swap `blob`
* replay tetap lolos hash

---

## 🔧 FIX WAJIB

Pastikan hash dihitung EXACT seperti ini:

```python
entry_hash = sha256(
    f"{seq}|{path}|{blob}|{prev_hash}".encode()
)
```

👉 Jangan pakai:

* timestamp (tidak deterministik)
* field opsional

---

# ⚠️ ISSUE-2: GENESIS WEAK POINT

```json

```

👉 Problem:

* chain tetap valid

---

## 🔧 FIX

```text
GENESIS_HASH = sha256("MIA_AS_JOURNAL_V1")
```

# ⚠️ ISSUE-4: SEQ GAP ATTACK


---

```python

```
